# JoystickRocker
这是一个使用Framelayout制作的摇杆控制器自定义控件。在xml中直接使用，目前版本需要提供大圆作为背景图片，小圆可提供图片或不提供。

![](https://github.com/luoyexk/JoystickRocker/blob/master/image/joystick-demo2.gif)
